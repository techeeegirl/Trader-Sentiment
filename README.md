# Trader Behavior vs Market Sentiment Analysis

This project analyzes how trader behavior on Hyperliquid aligns with Bitcoin market sentiment using the Fear & Greed Index.

## Structure
- notebook_1.ipynb: Full analysis in Google Colab
- csv_files/: Processed datasets
- outputs/: Visualizations (PNG)
- ds_report.pdf: Final summarized insights

## Key Insights
- Greed increases volume, leverage-like exposure, and emotional trading
- Fear reduces participation but improves trade quality
- Sentiment is a leading behavioral signal, not just descriptive

## Execution
All analysis was performed using Python (Pandas, Matplotlib, Seaborn) in Google Colab.
